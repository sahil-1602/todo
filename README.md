# _Todo App_

A simple todo app made using react and redux

[Vist Website](https://sahil-1602.github.io/todo/)

### Build With

- `React`
- `sass`
- `react-router`
- `react-hooks`
- `redux`
- `Fluent ui`

## INSTALLATION

fork or download the repository

```npm
npm install

cd todo
npm start
```
